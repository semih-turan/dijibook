## How to Use
`git clone https://github.com/semih-turan/dijibook.git`

`cd dijibook`

`npm install` 

**To Start Metro Server,**

`npm start run` 

**For android:** 

`npm run android`

**For ios:**

`cd dijibook`

`bundle install`

`cd dijibook/ios`

`pod install`

`cd ..`

`npm run ios`
